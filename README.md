# Inflearn-Items

이 프로젝트는 Inflearn 강의의 일환으로 개발된 간단한 상품 관리 애플리케이션입니다. Spring Boot와 Thymeleaf를 활용하여 기본적인 CRUD(Create, Read, Update, Delete) 기능을 구현했습니다.

## 특징

- 상품 목록 조회: 등록된 모든 상품의 목록과 상세 정보를 확인할 수 있습니다. 
- 상품 등록: 새로운 상품을 추가할 수 있습니다. 
- 상품 수정: 기존 상품의 정보를 수정할 수 있습니다. 
- 상품 삭제: 상품을 목록에서 삭제할 수 있습니다.

## 기술 스택

- 백엔드: Spring Boot, Spring MVC 
- 프론트엔드: Thymeleaf, Bootstrap
- 빌드 도구: Gradle 
- 언어: Java

## 사용법

- 상품 목록 조회: /basic/items 경로로 이동하여 상품 목록을 확인하세요. 
- 상품 등록: "상품 등록" 버튼을 클릭하여 새로운 상품을 추가하세요. 
- 상품 수정: 목록에서 수정할 상품을 선택하여 정보를 수정하세요. 
- 상품 삭제: "삭제" 버튼을 클릭하여 상품을 삭제하세요.

## 기타 정보

이 프로젝트는 학습 목적으로 제작되었으며, 간단한 CRUD 기능 구현을 중점으로 하고 있습니다. 추가적인 질문이나 피드백은 언제든지 환영합니다!
